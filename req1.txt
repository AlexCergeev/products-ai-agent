Требования по изменению формулы расчета максимального срока кредита:

1. Цель: Обновление формулы расчета максимального срока кредита для повышения гибкости и привлекательности кредитных продуктов.

2. Новая формула: Максимальный срок кредита = min(Максимальный возраст клиента на момент погашения - Возраст клиента, Максимальный срок по продукту, Ограничение по доходу).

3. Параметры формулы:
   - Максимальный возраст клиента на момент погашения: Устанавливается банком (например, 70 лет).
   - Возраст клиента: Возраст клиента на момент подачи заявки.
   - Максимальный срок по продукту: Ограничение, установленное для конкретного кредитного продукта (например, 30 лет).
   - Ограничение по доходу: Рассчитывается на основе ежемесячного платежа, который не должен превышать определенный процент от дохода клиента (например, 40%).

4. Условия:
   - Формула применяется для всех новых кредитных заявок.
   - Возможность ручного пересмотра срока в исключительных случаях (по решению кредитного комитета).

5. Документация:
   - Обновление внутренних регламентов и кредитной политики.
   - Информирование клиентов об изменениях через официальные каналы.

6. Автоматизация:
   - Интеграция новой формулы в кредитный калькулятор и системы автоматического одобрения заявок.
